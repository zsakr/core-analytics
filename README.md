## Overview

Core Analytics is a cutting-edge platform that revolutionizes squash performance analysis using artificial intelligence. Our platform provides real-time insights, performance metrics, and personalized recommendations for players and coaches.

## Features

- **AI-Powered Analysis**: Advanced machine learning algorithms for motion analysis
- **Real-time Tracking**: Instant feedback on player movements and techniques
- **Performance Dashboard**: Comprehensive analytics and progress tracking
- **Team Management**: Tools for coaches to manage multiple players
- **Mobile Responsive**: Seamless experience across all devices
- **Interactive Reports**: Detailed performance reports and recommendations

## Tech Stack

- **Frontend**:
  - HTML5, CSS3, JavaScript
  - AOS (Animate On Scroll)
  - Font Awesome Icons
  - Modern CSS Grid and Flexbox
  - Responsive Design

- **Design**:
  - Custom CSS Variables
  - Mobile-First Approach
  - Smooth Animations
  - Dark Theme Optimized

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/zsakr/core-analytics.git
   ```

2. Navigate to the project directory:
   ```bash
   cd core-analytics
   ```

3. Open `index.html` in your browser to view the website

## Development

- All styles are modularized in the `css` directory
- JavaScript functionality is organized in `js/main.js`
- Assets and images are stored in the `assets` directory
- The website is fully responsive and optimized for all screen sizes
